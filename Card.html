About This Project

The Card Creation Website is a mini-project built using HTML, CSS, and JavaScript. It allows users to generate customized cards by entering details like profile image URL, name, title, and description. Cards are saved automatically so they survive page refreshes.

This project demonstrates DOM manipulation, localStorage, live previews, and dynamic UI generation — all beginner-friendly concepts in web development.

  • Key Features
  • Live card preview as you type
  • Multiple color themes
  • Edit & delete cards
  • Download card as image
  • Cards saved with localStorage
  • Dark mode support
  • Future Enhancements
  • Drag-and-drop image upload
  • Multiple card templates
  • Export all cards as PDF
  • Share card via link
  • Card reordering
