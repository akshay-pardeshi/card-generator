# 🃏 Card Generator Web App

A simple and interactive **Card Generator Web Application** built using **HTML, CSS, and JavaScript**.  
This project allows users to dynamically generate stylish cards with custom content.

---

## 🚀 Features

- 🎨 Generate dynamic cards instantly  
- ✏️ Custom input for card content  
- 📱 Responsive design (Mobile + Desktop)  
- ⚡ Fast and lightweight  
- 🎯 Clean UI with modern styling  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## ⚙️ How to Run

1. Clone the repository
2. Open the folder  
3. Run `index.html` in your browser
